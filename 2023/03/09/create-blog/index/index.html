初始化博客

一、环境

hexo 需要安装 node.js 和 GitHub

二、安装

三、初始化

下载 hexo ：hexo init <folder>
下载依赖 : npm install
生成 public 文件夹 : hexo generate (简写 hexo g)
启动 : hexo server (简写 hexo s)

四、美化

安装 Fliud 主题

修改 _config.yml 文件

注意：hexo v5.0 之后使用 theme,会出现{% extends '_layout.swig' %} {% import '_macro/post.swig' as post_template %}问题
原因 : hexo v5.0 之后删除了 swig ，需要手动安装 npm i hexo-renderer-swig

补充 : hexo 官网提供的安装 Fluid 主题方法，参考
